# بوت تيليجرام — النسخة المعتمدة لتكامل ZaFeer (Read-Only + CSV + Alerts)

> **آخر تحديث**: 2026-05-21
> **وضع التشغيل المعتمد**: قراءة فقط (لا إضافة، لا تعديل، لا حذف)

---

## 1) الهدف

البوت مساعد سريع للاستعلام والتنبيهات فقط.

- يستقبل أوامر استعلام
- يرجع النتائج مختصرة داخل الرسالة
- إذا النتائج كبيرة: يرسل ملف CSV
- يرسل تنبيهات يومية حسب الإعدادات

**ممنوع تمامًا من البوت:**
- إضافة بيانات
- تعديل بيانات
- حذف بيانات
- اعتماد رواتب
- تغيير حالة مستخلصات

---

## 2) نموذج الصلاحيات

### حساب البوت داخل النظام

نعم، يتم إنشاء مستخدم فعلي باسم واضح مثل:
- `telegram.bot@zafeer.local`

هذا المستخدم يكون له صلاحيات مثل أي مستخدم، لكن دور مقيّد للقراءة فقط.

### من يقدر يراسل البوت

مسموح فقط لـ:
- أنت
- شخص موثوق واحد

يتم التحقق برقم `Chat ID` فقط، وأي رقم آخر يتم تجاهله.

---

## 3) تسجيل الأحداث (مثل أي مستخدم)

كل أمر من البوت لازم يتسجل في السجل مثل أي نشاط داخل النظام.

كل سجل يحتوي:
- المستخدم الداخلي: `telegram.bot`
- رقم المرسل في تيليجرام
- اسم المرسل في تيليجرام
- نوع العملية (استعلام/تصدير CSV/تنبيه)
- وقت التنفيذ
- النتيجة (نجاح/فشل)

بهذا لا يوجد نشاط مجهول.

---

## 4) القنوات المتاحة للتنبيهات (Checkbox)

كل نوع تنبيه له 3 اختيارات مستقلة:

- [ ] داخل التطبيق
- [ ] إيميل
- [ ] تيليجرام

يمكن تشغيل أي قناة وحدها أو تشغيل الثلاثة معًا.

---

## 5) سياسة النتائج الطويلة

- لو النتيجة قصيرة: تظهر كنص داخل تيليجرام
- لو النتيجة طويلة: تتحول تلقائيًا إلى CSV

**صيغة موحدة للملفات:**
- UTF-8
- فاصل أعمدة: `,`
- تاريخ: `dd/MM/yyyy`
- تسمية الملفات مثال:
  - `employees_2026-05-21_1030.csv`
  - `payroll_2026-05.csv`

## 7) الرواتب والمستخلصات في البوت

### الرواتب
- البوت لا يعتمد الرواتب
- البوت فقط يجيب بيانات المسير ويرسل CSV عند الطلب

### المستخلصات
- البوت لا يغيّر حالة المستخلص إلى مدفوع
- البوت فقط يجيب بيانات المستخلصات ويرسل CSV عند الطلب

---

## 8) البحث العربي

لتحسين النتائج في الأسماء العربية:

- تجاهل التشكيل
- اعتبار (أ/إ/آ) مثل (ا)
- اعتبار (ى) مثل (ي)
- السماح ببحث جزئي من أي جزء في الاسم

الهدف: لو كتبت الاسم بشكل مختلف بسيط، يطلع نفس الشخص.

---

## 9) الأمان

- السماح لرقمين فقط (أنت + شخص موثوق)
- Rate limit لكل Chat ID
- فحص `secret_token` للـ webhook
- إخفاء البيانات الحساسة عند العرض
- منع أي أوامر كتابة من الكود نفسه

---

## 10) منع تكرار نفس الطلب

أحيانًا نفس الرسالة قد تصل مرتين بسبب الشبكة.

الحل:
- لكل رسالة رقم فريد من تيليجرام
- إذا الرقم اتنفذ قبل كده: يتم تجاهله

النتيجة: لا تكرار في التنفيذ.

---

## 11) التوقيت (السعودية)

الاعتماد الرسمي للتنبيهات:
- توقيت السعودية `Asia/Riyadh`

تنفيذ الجدولة يكون بتوقيت السيرفر (UTC) بعد التحويل.

مثال:
- 07:00 صباحًا السعودية = 04:00 UTC

## 12) الجداول المطلوبة

```sql
CREATE TABLE telegram_whitelist (
  chat_id BIGINT PRIMARY KEY,
  user_id UUID REFERENCES users(id),
  enabled BOOLEAN DEFAULT true,
  created_at TIMESTAMPTZ DEFAULT now()
);

CREATE TABLE telegram_command_log (
  id BIGSERIAL PRIMARY KEY,
  chat_id BIGINT,
  telegram_name TEXT,
  command TEXT,
  args TEXT,
  result_status TEXT, -- success | error | unauthorized
  created_at TIMESTAMPTZ DEFAULT now()
);

CREATE TABLE telegram_user_prefs (
  chat_id BIGINT PRIMARY KEY,
  lang TEXT DEFAULT 'ar',
  notify_in_app BOOLEAN DEFAULT true,
  notify_email BOOLEAN DEFAULT true,
  notify_telegram BOOLEAN DEFAULT true,
  created_at TIMESTAMPTZ DEFAULT now()
);

CREATE TABLE telegram_dedup (
  update_id BIGINT PRIMARY KEY,
  processed_at TIMESTAMPTZ DEFAULT now()
);
```

## 15) ما لن نفعله في هذا المشروع

- لا اعتماد رواتب من تيليجرام

- لا PDF من البوت (CSV فقط)
- لا إنشاء/تعديل/حذف أي بيانات من البوت

## 16) أسئلة بسيطة تحتاج ردك (بدون تعقيد)

1. ما هما اسما الشخصين المسموح لهما فقط باستخدام البوت؟
2. هل البوت يكون في محادثة خاصة فقط، أم مسموح أيضًا في جروب؟
3. ما هو وقت التنبيه اليومي الذي تريده بالضبط بتوقيت السعودية؟
4. هل تريد تنبيه صباحي فقط، أم صباحي ومسائي؟
5. في تنبيهات اليوم، ما أهم 5 أشياء تريدها تظهر أولًا؟
6. عند طلب مسير الرواتب CSV، هل تريد كل الموظفين أم ملخص فقط؟
7. عند طلب مستخلصات CSV، هل تريد كل الحالات أم فقط المتأخر؟
8. هل تريد أن أسماء الأعمدة في CSV تكون بالعربي أم بالإنجليزي؟
9. أقصى عدد صفوف في الملف الواحد: هل نرسله ملف واحد كبير، أم نقسمه لأكثر من ملف؟
10. هل تريد البوت يرد بالعربي فقط، أم عربي وإنجليزي؟
11. هل تريد حفظ نسخة من كل CSV داخل النظام للرجوع لها لاحقًا؟
12. إذا لم توجد نتائج للاستعلام، تفضّل رسالة قصيرة أم ملف CSV فارغ؟
13. هل تريد إضافة أمر سريع مثل: "أرسل لي تقرير اليوم" بأمر واحد؟
14. هل تريد إخفاء الراتب دائمًا في الرسائل النصية، وإظهاره فقط داخل CSV؟
15. هل في بيانات لا تريد ظهورها أبدًا في تيليجرام حتى لو للمستخدمين المسموح لهم؟
